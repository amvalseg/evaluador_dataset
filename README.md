# DataScope — Dataset Evaluation Tool

**DataScope** es una herramienta web ligera para evaluar la **calidad y madurez de datasets**, especialmente orientada a estudiantes de análisis de datos.

🌐 **Demo:** [https://datascope.humanidata.es](https://datascope.humanidata.com)

---

## Qué hace

DataScope permite:

- Evaluar datasets en múltiples dimensiones:
  - Estructura
  - Calidad
  - Potencial pedagógico (en función de las tareas que permite realizar)
- Obtener una **valoración rápida y visual**

> Todo ocurre **en el navegador**. No se envían datos a ningún servidor.

---

## Enfoque

Esta herramienta nace de una necesidad práctica:

> Para elaborar un buen portfolio de análisis de datos, es un reto seleccionar **conjuntos de datos con potencial analítico pero también pedagógico**.

DataScope busca cerrar ese gap de forma:

- **Simple**
- **Accesible**
- **Accionable**

---

## Cómo usarlo

1. Accede a la herramienta
2. Sube el archivo/s a analizar
3. Revisa el resultado
4. Si quieres guardarlo, exporta el resultado en PDF

> No requiere instalación.

---

## Stack tecnológico

- **HTML**
- **CSS**
- **JavaScript (vanilla)**

> Sin dependencias externas ni backend.

---

## Privacidad

- No se recogen datos
- No hay tracking
- Todo el procesamiento es **local**
